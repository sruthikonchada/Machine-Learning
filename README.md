# Machine-Learning
Machine Learning Algorithms Practice:
This repository contains implementation of basic supervised machine learning algorithms using Python and Scikit-Learn with real time datasets.
**Algorithms Implemented**
**1.Linear Regression**
Predicts continuous output
Implemented using Scikit-Learn
Includes data visualization and model evaluation
**2. Multiple Linear Regression**
Uses multiple independent variables
Demonstrates feature impact on target variable
Model performance evaluated using R² score
**3️. Logistic Regression**
Used for classification problems
Predicts categorical output (0 or 1)
Includes accuracy evaluation
**Technologies Used**:
Python
NumPy
Pandas
Matplotlib
Scikit-learn
Google Colab
